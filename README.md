# 🤖 Remnasale
<sub>Telegram-бот для продажи VPN-подписок с интеграцией Remnawave Panel</sub>

> **Версия:** `0.0.1`
___

## 🚀 Быстрый старт

### ▶️ Полная установка одной командой
```bash
cd /opt && bash <(curl -s https://raw.githubusercontent.com/DanteFuaran/Remnasale/main/remnasale-install.sh)
```

### 📝 Данные для настройки

Скрипт запросит следующие параметры:

> **Домен бота** - домен для webhook (например, `bot.example.com`)<br>
> **Токен бота** - получите в [@BotFather](https://t.me/BotFather)<br>
> **Telegram ID разработчика** - ваш числовой ID (используйте [@userinfobot](https://t.me/userinfobot))<br>
> **Username поддержки** - никнейм группы/канала поддержки (без `@`)<br>
> **API токен Remnawave** - токен из панели Remnawave с правами администратора

<sub>⚠️ $${\color{orange}Ключи\ шифрования,\ пароли\ БД\ и\ Redis\ генерируются\ автоматически.}$$</sub>

### 📦 Что сделает скрипт
> ✅ Установит Docker и Docker Compose<br>
> ✅ Создаст структуру каталогов<br>
> ✅ Сгенерирует конфигурацию `.env`<br>
> ✅ Развернет PostgreSQL и Redis<br>
> ✅ Запустит бота и применит миграции БД<br>
> ✅ Настроит webhook для Telegram<br>
> ✅ Установит панель управления `remnasale`

<sub>После установки управление доступно через команду **`remnasale`**</sub>

<h2> $${\color{red}🔧 Частые\ ошибки}$$ </h2>

<details>

<summary>$${\color{gray}Показать\ подробности:}$$<br></summary>

### Убедитесь что:
> ✅ Домен бота правильно указывает на IP сервера через **A-запись**<br>
> ✅ **Токен бота** корректный и активный (проверьте в @BotFather)<br>
> ✅ **Payments Provider** подключен для использования Telegram Stars<br>
> ✅ **API токен Remnawave** имеет права администратора<br>
> ✅ **URL панели Remnawave** доступен с сервера бота<br>
> ✅ Порты **80** и **443** свободны и не заняты другими службами<br>
> ✅ Firewall разрешает входящие подключения на портах **80**, **443**

### Решение проблем:
> 📜 Проверьте логи через панель управления `remnasale`<br>
> 🔄 Попробуйте перезапустить бота через панель управления<br>
> ⚙️ Проверьте настройки в `.env` (команда `⚙️ Изменить настройки`)<br>
> 🔍 Убедитесь что Remnawave Panel доступна и работает<br>
> 🆘 Если проблема не решается - откройте **Issue** на GitHub с логами

</details>
